# OrbDash

OrbDash is a windows playable game where you play as a orb/circle and jump over obstacles and use abilities.

## Features
- 7 built in levels
- level editor
- boost
- best times
- skins

## How to play
Controls :
  SPACE or mouse click - jump / start / continue
  left/right SHIFT - hold to boost speed
  esc - back to level select

## How to download
1. Download the zip file from releases
2. Extract/unzip the folder anywhere on your PC
3. Open the extracted folder and double click OrbDash.exe to play

If Windows SmartScreen blocks it, click "More info" then "Run anyway".

## Tech stack
Built in C++ using SFML
Compiled with g++

## System requirements
Windows 10 or 11
